# Инструкция по подключению Google Search Console

Чтобы ваш сайт появился в поиске Google, выполните следующие шаги:

## 1. Регистрация в Google Search Console
1. Перейдите на [Google Search Console](https://search.google.com/search-console/).
2. Введите URL вашего сайта: `https://client.eelepkal.com/`.

## 2. Подтверждение прав на домен
Google попросит вас подтвердить, что вы владелец сайта. Есть два основных способа:

### Способ А: HTML файл (рекомендуется)
1. Скачайте проверочный HTML-файл из панели Google.
2. Попросите разработчика (или загрузите сами) положить этот файл в папку `public/` вашего проекта.
3. Разверните проект (deploy).
4. Нажмите "Verify" в Google Search Console.

### Способ Б: DNS-запись
1. Скопируйте предоставленную TXT-запись.
2. Добавьте её в настройки DNS вашего домена (там же, где вы настраивали A-записи для Vercel).

## 3. Добавление Sitemap
После подтверждения прав:
1. В левом меню выберите раздел **Sitemaps**.
2. В поле "Add a new sitemap" введите: `sitemap.xml`.
3. Нажмите **Submit**.

## 4. Запрос на индексацию конкретных страниц
Если вы хотите ускорить появление страницы `/venues` в поиске:
1. Вставьте URL `https://client.eelepkal.com/venues` в строку поиска в самом верху панели GSC.
2. Нажмите **Request Indexing**.

## 5. Проверка Robots.txt
Убедитесь, что Google видит ваш файл: `https://client.eelepkal.com/robots.txt`. 
Там должно быть написано `Allow: /`.

---
**Совет**: Индексация может занять от нескольких дней до нескольких недель. Проверяйте раздел "Coverage", чтобы увидеть, какие страницы уже попали в базу Google.
